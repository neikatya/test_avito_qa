# Ads Microservice API Tests

## Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/neikatya/avito-qa-internship-test
cd avito-qa-internship-test

# Установите зависимости
pip install -r requirements.txt
```

---

## Запуск тестов

```bash
pytest -v
```

---

## Описание тестовых модулей

### test_create_item.py
Тестирование создания объявлений (TC-1.1 - TC-1.6):
- Успешное создание с валидными данными
- Отсутствие обязательных полей
- Невалидные типы данных
- Граничные значения и edge cases

### test_get_item.py
Тестирование получения объявления по ID (TC-2.1 - TC-2.3):
- Получение существующего объявления
- Несуществующий ID
- Невалидный формат ID

### test_get_seller_items.py
Тестирование получения списка объявлений продавца (TC-3.1 - TC-3.3):
- Продавец с несколькими объявлениями
- Продавец без объявлений
- Невалидный sellerID

### test_statistics.py
Тестирование получения статистики (TC-4.1 - TC-4.4):
- Получение статистики через v1 и v2 API
- Сравнение ответов v1 и v2
- Несуществующее объявление
- Невалидный ID

### test_delete_item.py
Тестирование удаления объявлений (TC-5.1 - TC-5.4):
- Успешное удаление
- Проверка консистентности после удаления
- Удаление несуществующего объявления
- Невалидный ID

### conftest.py
Содержит общие фикстуры:
- `base_url` - базовый URL API
- `valid_item_payload` - валидные данные для создания объявления
- `created_item` - фикстура, создающая объявление для тестов
- `created_items_for_seller` - фикстура, создающая несколько объявлений для одного продавца
- Утилиты: `extract_item_id()`, `generate_unique_seller_id()`
